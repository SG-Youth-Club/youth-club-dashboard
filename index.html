🕒 SESSION TIMELINE

5:45pm

Staff arrive and discuss who's attending the youth club, and set up the centre based on who's coming.

6:10pm

Team chat - discuss plan and staff roles for the session, who's going on the door, who's upstairs, who's downstairs etc.

6:30pm

Young people arrive.

7:50pm

Start preparing for the end of the session. One member of staff should be answering the door and another member of staff at the top of the stairs calling the names of any YP upstairs.

8:00pm

Youth club finishes, young people leave. Staff tidy the centre and reflect on the session before leaving.

8:45pm

Staff leave.

🛠 SETUP TASKS

  • Centre to be set up and ready for youth club - All staff
  • VR headset and Switches to be charged and set up
  • Check and follow youth centre checklist to make sure the centre is set up correctly. This can be found at the top of this page or in the storage room in the kitchen.

🏠 ROOM SETUP

Downstairs Room
Make sure consoles, PC and board games are out and set up. Make sure the room is tidy and that the session information is on the whiteboard.
Gaming Room
Make sure all consoles and PCs are set up and a game is preloaded.
Quiet Room
Set up activities, can be anything: art, music, board games. Just make sure it is a quiet space.
Sensory Room
Make sure lights are dimmed and all LEDs are charged and that the room is being used properly.
Kitchen Area
Make sure drinks and snacks are set up, and that there is also an activity set up.

🍪 SNACKS & DIETARY

Snacks and drinks should be set up and available throughout the session. We should have out the following:

  • Biscuits and snacks
  • Water available at all times
  • Juice and soft drinks
  • Check any dietary requirements, as some YP may be allergic or intolerant

    🚗 TRAVEL & PARKING

    • If travelling by public transport, the Bury Interchange and Bury Metrolink are only a 5-minute walk away.
    • The community centre itself does not have any parking; however, there are a number of places available only a short walk away.
